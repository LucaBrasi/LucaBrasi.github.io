Functions

return-type function-name(parameter-list);

First we need function prototype: 

    int add_values (int x, int y);

Then we introduce function body:

    int add_values (int x, int y)
{
        return x + y;
}

Now we can start using our sweet function. Call it in your program like so:

    int main(void)
{
        int x = 1, y = 2;
        add_values(x, y);
        return 0;
}
